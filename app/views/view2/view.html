Tabs

I took a lot of pride and recevied great feedback for these AJAX'd and cached tabs I built for Wiley. Part of the specification was to return the user to the exact point they had navigated from…

I wrote a façade module for localstorage which also came in really useful down the road for other tasks.

Using localstorage I saved the state of the current tab, content and scroll position. This plays nicely with the 'more' link functionality contained in the tabs also.

One think I notice sites rarely take you back to the correct state after navigating away from content dynamically loaded. I was very conscious of this whilst developing.

Custom site development

I worked on a complete custom build for a site called Ozy whilst at Abacus.

This was a very exciting project for me as it not only pushed me further than any other project but helped me grow as a developer. Not to mention the oppurtunity to work out in Mountain View, California also.

The site was also featured as part of the 'A Responsive Web Design Podcast'

For the initial phase I worked alongside a Designer, Developer and Project Manager giving advice on frontend development, timescales, prototyping concepts and building the base code for the project.

I returned to the UK to work alongside a wider team of Frontend Developers to complete the project.

Naturally over time the site has been refactored and redeveloped but the homepage has stayed relatively true to my original build and it's still a site I feel proud to have been a part of.
